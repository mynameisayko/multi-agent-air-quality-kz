# Multi-Agent System with Temporal Fusion Transformer for Urban Air Quality Forecasting and Health Risk Assessment in Kazakhstan

Private research workspace for code, manuscript drafting, experiments, and project documentation.

## Recommended workflow

- `src/` for reusable code
- `notebooks/` for exploration and one-off analysis
- `manuscript/` for article drafts, figures, and submission materials
- `configs/` for experiment settings
- `data/` for local datasets only
- `reports/` for outputs and presentations
- `docs/` for collaboration and project decisions

## Collaboration model

Use the Git repository for:

- source code
- manuscript text
- configs
- metadata
- small processed tables
- documentation

Do not store large raw datasets or sensitive health-related data directly in GitHub. Keep them in shared cloud storage and track only:

- dataset descriptions
- download or access instructions
- schema files
- processing scripts
- checksums if needed
