# Data Management

This directory is for local data handling only.

## Tracked in Git

- folder structure
- dataset documentation
- schemas
- lightweight sample files if they are safe to share

## Not tracked in Git

- raw datasets
- private medical or health records
- large processed tables
- binary model artifacts

## Suggested shared storage

Create a cloud folder with:

- `01_raw_data`
- `02_processed_data`
- `03_models`
- `04_shared_outputs`

Keep a short source and access note for each dataset in this repository.
